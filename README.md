## 🌱 What is Verdex?

Verdex is a **comprehensive environmental intelligence platform** that democratizes access to climate data and action. Our platform combines:

- **🛰️ Real-time environmental monitoring** from NASA, NOAA, and global sensor networks
- **🧠 AI-powered data analysis** with machine learning models and GPT-4 explanations  
- **🎯 Gamified citizen science** with location-based environmental missions
- **📋 Policy generation tools** for evidence-based environmental decision making

**No registration required** - Start exploring environmental data and taking climate action immediately.

## 🎯 Our Mission

**"Empower people worldwide to understand and act on environmental challenges through accessible data, AI insights, and community collaboration."**

## 🛠️ Technology Stack

### Frontend & User Experience
- **Framework**: Next.js 14 with App Router for optimal performance
- **Styling**: Tailwind CSS with forest-inspired, accessible design
- **Maps**: Interactive environmental mapping with real-time data layers
- **Charts**: Advanced data visualization for environmental trends
- **Mobile-first**: Fully responsive design for all devices

### Data & Intelligence
- **Real-time APIs**: NASA EarthData, NOAA, AQICN, OpenWeatherMap
- **AI Integration**: GPT-4 for natural language explanations and insights
- **Machine Learning**: Custom models for environmental prediction and analysis
- **Data Processing**: Advanced algorithms for environmental data interpretation

### Infrastructure
- **Database**: PostgreSQL for reliable data storage

- **Security**: Enterprise-grade security without compromising accessibility
- **Scalability**: Built to handle millions of environmental data points

2. **Explore the Dashboard**: View real-time environmental data for any location

### For Developers

\`\`\`bash
# Clone the repository
git clone https://github.com/verdex-platform/verdex.git
cd verdex

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your API keys for environmental data sources

# Start development server
npm run dev

# Open http://localhost:3000
\`\`\`

### Environment Variables
\`\`\`env
# Environmental Data APIs
AQICN_API_KEY="your_air_quality_api_key"
OPENWEATHER_API_KEY="your_weather_api_key"
NASA_API_KEY="your_nasa_api_key"

# AI Services
OPENAI_API_KEY="your_openai_api_key"

# Maps
NEXT_PUBLIC_MAPBOX_TOKEN="your_mapbox_token"

# Database
DATABASE_URL="your_database_url"
\`\`\`
